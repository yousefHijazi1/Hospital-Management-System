Installation:

Clone the Repo:

- git clone hhttps://github.com/yousefHijazi1/Hospital-Management-System.git
- composer install or composer update
- cp .env.example .env
- Set up .env file
- php artisan key:generate
- php artisan migrate
- php artisan serve
